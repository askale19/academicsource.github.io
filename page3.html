About Us

As technology advances internet is becoming ever popular. Website has become an essential part. Often website of an institution is used as major requirements to rate it. Thus having a website is essential for any major organization. As It is educational website to make study soft and smooth way. It is best technology to overview the online achievements of students. It is well medium to interact with students, teachers and Parents under one roof. It is html based website including all data of every individual students, Score of all tests, practical’s, assignments and submission so on. It is not only shows academic information but also student’s growth through it. Notice, research works, club activities, digital local library etc. can be the major advantages of a website.
There are five student’s groups worked on this project who already having enough knowledge about web designing. We all had to learn some programming languages which are necessary to develop well website. Two members worked as planning and implementation which need to make it possible. They also work on making Power point and report writing on it. Other two or three worked as frontend and backed programmer in this project. Html, CSS and JS are playing crucial role in developing and working this website.
It is web-application named as AcademicInfo which provides all students’ data in it. It is very helpful technology for teacher and time consuming way to archive all academic information. It is required because Online Learning is necessary in corona pandemic. Every Educational Organization has its own website. But It only gives information about that institute. It is necessary to have well equipped website which archives data of daily tasks occurring in institute. AcademicInfo also provides access of lectures, notes and online books which make it online classroom for students. Every Individual can get any academic information of oneself regarding test scores, Question papers, Recorded video of Lectures and practical’s both through it. It’s features make it very sophisticated and enthusiastic. It is well platform for student to achieve well academic growth in his life.
